The problem

Nepal's government services are scattered across hundreds of .gov.np websites — many in legacy fonts (Preeti, Kantipur), behind PDFs no search engine indexes, with procedural details that change ward-by-ward.

Hello Sarkar 1111 helps over the phone, but is unreachable from a queue at 9 AM. Search engines return form templates, not "which counter do I go to first." The practical knowledge — the thing every experienced agent knows in their head — rarely makes it onto a webpage.

What we built

877

Gov.np sources crawled

A custom Rust crawler keeps a living corpus of federal, provincial, and palika sites — Devanagari + Preeti font conversion, PDF text extraction, deduplication, freshness scoring.

growing

Tacit corpus from interviews

Citizens, ward officers, and consultants record what they know about the offices they work — counter routes, timing, paperwork pitfalls. Stored as immutable audio, curated by an admin before joining the corpus.

Gemma 4

Fine-tuned LoRA composer

Built on Google's Gemma 4 E2B, fine-tuned with a refusal-when-no-source discipline and bilingual citation grammar. Composer over a hybrid retrieval pipeline — not a free-form chatbot.

cite

Or refuse

Every factual claim carries the source URL it came from. If no source meaningfully addresses your question, the system says so — and tells you to call Hello Sarkar 1111.

How it works

Citizen path Question → retrieval over both corpora → composed answer with citations. Public, no auth.
Interview path Anyone records short answers per question. Audio is the source of truth. Admin reviews, transcribes via Vertex, then promotes to corpus.
Where it runs The model lives on a Mac Studio (M2 Ultra) over Tailscale. A Hetzner reverse-proxy on the public domain.

Stack

  • CrawlerRust + SQLite + custom recipe-driven crawler over 877 sources
  • RetrievalSQLite FTS5 + bilingual anchor expansion (English/Roman-NE → Devanagari)
  • ComposerGemma 4 E2B-IT + LoRA adapter trained on grounded-answer / refusal / anti-template slices
  • ServerFastAPI on Mac Studio M2 Ultra · transformers + MPS
  • Tacit ASRVertex AI (Gemini 2.5) for transcription on admin approve
  • HostingTraefik + Let's Encrypt on Hetzner Cloud → tailnet → k2
